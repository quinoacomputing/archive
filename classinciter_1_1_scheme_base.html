inciter::SchemeBase::SchemeBase ( ctr::SchemeType  scheme)
inlineexplicit

Constructor

Parameters
[in]schemeDiscretization scheme

Based on the enum we create two empty chare arrays: (1) discproxy which contains common functionality and data for all discretizations, and (2) proxy, which have functionality and data specific to a given discretization. Note that proxy is bound (in migration behavior and properties) to discproxy.
