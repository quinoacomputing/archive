2 /*!
3  \file src/Inciter/DiagCG.h
4  \copyright 2012-2015, J. Bakosi, 2016-2018, Los Alamos National Security, LLC.
5  \brief DiagCG for a PDE system with continuous Galerkin without a matrix
6  \details DiagCG advances a system of partial differential equations (PDEs)
7  using continuous Galerkin (CG) finite element (FE) spatial discretization
8  (using linear shapefunctions on tetrahedron elements) combined with a time
9  stepping scheme that is equivalent to the Lax-Wendroff (LW) scheme within
10  the unstructured-mesh FE context and treats discontinuities with
11  flux-corrected transport (FCT). The left-hand side (lumped-mass) matrix
12  is diagonal thus this scheme does not use a matrix-based linear solver.
13 
14  There are a potentially large number of DiagCG Charm++ chares created by
15  Transporter. Each DiagCG gets a chunk of the full load (part of the mesh)
16  and does the same: initializes and advances a number of PDE systems in time.
17 
18  The implementation uses the Charm++ runtime system and is fully
19  asynchronous, overlapping computation and communication. The algorithm
20  utilizes the structured dagger (SDAG) Charm++ functionality. The high-level
21  overview of the algorithm structure and how it interfaces with Charm++ is
22  discussed in the Charm++ interface file src/Inciter/diagcg.ci.
23 
24  #### Call graph ####
25  The following is a directed acyclic graph (DAG) that outlines the
26  asynchronous algorithm implemented in this class The detailed discussion of
27  the algorithm is given in the Charm++ interface file transporter.ci, which
28  also repeats the graph below using ASCII graphics. On the DAG orange
29  fills denote global synchronization points that contain or eventually lead
30  to global reductions. Dashed lines are potential shortcuts that allow
31  jumping over some of the task-graph under some circumstances or optional
32  code paths (taken, e.g., only in DEBUG mode). See the detailed discussion in
33  diagcg.ci.
34  \dot
35  digraph "DiagCG SDAG" {
36  rankdir="LR";
37  node [shape=record, fontname=Helvetica, fontsize=10];
38  OwnLhs [ label="OwnLhs"
39  tooltip="own contributions to the left hand side lumped-mass
40  matrix computed"
41  URL="\ref inciter::DiagCG::lhs"];
42  ComLhs [ label="ComLhs"
43  tooltip="contributions to the left hand side lumped-mass matrix
44  communicated"
45  URL="\ref inciter::DiagCG::comlhs"];
46  OwnRhs [ label="OwnRhs"
47  tooltip="own contributions to the right hand side computed"
48  URL="\ref inciter::DiagCG::rhs"];
49  ComRhs [ label="ComRhs"
50  tooltip="contributions to the right hand side communicated"
51  URL="\ref inciter::DiagCG::comrhs"];

53  tooltip="own contributions to the mass diffusion rhs computed"
54  URL="\ref inciter::DiagCG::rhs"];
55  ComDif [ label="ComDif"
