Parameters
[in]connVector of mesh element connectivity owned (global IDs) mesh chunk we operate on
[in]nbfac_completeTotal number of boundary-faces (triangles) in the entire mesh
[in]bface_completeMap of boundary-face lists mapped to corresponding side set ids for the entire mesh
[in]triinpoel_completeInterconnectivity of points and boundary-face in the entire mesh

This class is created per chare-worker in Partitioner::createWorkers(). This is done so that Discretization will not hold all this data unnecessarily, viz. for MatCG and DiagCG, where it's not needed. It will be computed only when DG discretization is chosen.
