Detailed Description

Keyword information bundle.

This bundle contains the information that is used to display on-screen help on all command-line arguments and control file keywords for an exectuable. This struct is stored in a container that associates keywords (used by a grammar and parser) to this struct. The container, an runtime, std::map, is filled by the CmdLine and InputDeck objects' constructors by one or more boost::mpl::for_each which loops through the set of all keywords used in a grammar. The maps are stored in the CmdLine and InputDeck objects (which are tagged tuples) and thus can be migrated through the network, thus the Charm++ parck/unpack routines are defined.
