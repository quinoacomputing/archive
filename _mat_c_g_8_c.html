Detailed Description

MatCG for a PDE system with continuous Galerkin with a matrix.

Copyright
2012-2015, J. Bakosi, 2016-2018, Los Alamos National Security, LLC.

MatCG advances a system of partial differential equations (PDEs) using continuous Galerkin (CG) finite element (FE) spatial discretization (using linear shapefunctions on tetrahedron elements) combined with a time stepping scheme that is equivalent to the Lax-Wendroff (LW) scheme within the unstructured-mesh FE context and treats discontinuities with flux-corrected transport (FCT). The left-hand side matrix is stored in a compressed sparse row (CSR) storage and thus uses a matrix-based linear solver.
