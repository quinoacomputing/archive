Detailed Description

template<class eq>
struct inciter::deck::check_transport< eq >

Set defaults and do error checking on the transport equation block.

This is error checking that only the transport equation block must satisfy. Besides error checking we also set defaults here as this block is called when parsing of a transport...end block has just finished.
