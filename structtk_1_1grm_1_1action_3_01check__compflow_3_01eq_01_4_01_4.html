Detailed Description

template<class eq>
struct tk::grm::action< check_compflow< eq > >

Set defaults and do error checking on the compressible flow equation block.

This is error checking that only the compressible flow equation block must satisfy. Besides error checking we also set defaults here as this block is called when parsing of a compflow...end block has just finished.
