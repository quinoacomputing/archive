Partition mesh using Zoltan2 with a geometric partitioner, such as RCB, RIB.

Parameters
[in]algorithmPartitioning algorithm type
[in]centroidMesh element coordinates
[in]elemidGlobal mesh element ids ] nelem Number of elements in mesh (on this MPI rank)
[in]npartNumber of desired graph partitions
Returns
Array of chare ownership IDs mapping graph points to concurrent async chares

This function uses Zoltan to partition the mesh graph in parallel. It assumes that the mesh graph is distributed among all the MPI ranks.
