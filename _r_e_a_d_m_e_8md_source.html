1 ## What is Quinoa?
2 
3 <img src="https://quinoacomputing.github.io/quinoa/quinoa.svg" align="right" width="25%" background=transparent>
4 Quinoa is a set of computational tools that enables research and numerical analysis in fluid dynamics. At this time it is a test-bed to experiment with various algorithms using fully asynchronous runtime systems.
5 
6 ## Organization
7 
8 Currently, Quinoa consists of the following tools:
9  - [<B>inciter</B>](http://quinoacomputing.github.io/quinoa/inciter_doc.html) - Partial differential equations solver on 3D unstructured grids
10  - [<B>walker</B>](http://quinoacomputing.github.io/quinoa/walker_doc.html) - Random walker for stochastic differential equations
11  - [<B>rngtest</B>](http://quinoacomputing.github.io/quinoa/rngtest_doc.html) - Random number generator test suite
12  - [<B>unittest</B>](http://quinoacomputing.github.io/quinoa/unittest_doc.html) - Unit test suite
13  - [<B>meshconv</B>](http://quinoacomputing.github.io/quinoa/meshconv_doc.html) - Mesh file converter
14 
15 ## Try
16 
17 The quickest way to try Quinoa is to run the already built executables inside the [release](https://hub.docker.com/r/quinoacomputing/quinoa/) [docker](https://www.docker.com) container.
18 
19 #### 1. Run the container on your local machine
20 
21 ```
22 docker run -ti quinoacomputing/quinoa:alpine
23 ```
24 
25 #### 2. Run executables inside the container, e.g.,
26 
27 ```
28 charmrun +p4 /usr/local/bin/unittest -v
29 ```
30

33 ## Build
34 
35 #### 1. Install prerequisites
36 
37 - Debian/Ubuntu linux: (line 1: required, line 2: recommended)
38 
39  ```
40  apt-get install cmake gfortran gcc g++ openmpi-bin libopenmpi-dev
41  apt-get install gmsh libpugixml-dev libpstreams-dev libboost-all-dev liblapack-dev liblapacke-dev libhdf5-dev libhdf5-openmpi-dev libhypre-dev
42  ```
43 
44 - Mac OS X: (line 1: required, line 2: recommended)
45 
46  ```
47  port install cmake openmpi-clang38 && port select clang mp-clang-3.8 && port select mpi openmpi-clang38-fortran
48  port install gmsh pugixml boost
49  ```
50 
51 #### 2. Clone, build third-party libraries, build & test
52 
53  ```
54  git clone https://github.com/quinoacomputing/quinoa.git; cd quinoa
55  mkdir tpl/build; cd tpl/build; cmake -DCMAKE_CXX_COMPILER=mpicxx -DCMAKE_C_COMPILER=mpicc -DCMAKE_Fortran_COMPILER=mpif90 ..; make; cd -
56  mkdir build; cd build; cmake -DCMAKE_CXX_COMPILER=mpicxx -DCMAKE_C_COMPILER=mpicc ../src; make; ../script/run_tests.sh
57  ```
58 
59  - All executables will be in <tt>quinoa/build/Main</tt>
