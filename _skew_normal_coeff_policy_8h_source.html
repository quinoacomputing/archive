18  const std::vector< kw::sde_T::info::expect::type >& timescale_,

22  std::vector< kw::sde_sigmasq::info::expect::type >& sigmasq,
23  std::vector< kw::sde_lambda::info::expect::type >& lambda )
24  \endcode
25  where
26  - ncomp denotes the number of scalar components of the system of the
27  skew-normal SDEs.
28  - Constant references to timescale_, sigmasq_, and lambda_, which denote
29  three vectors of real values used to initialize the parameter vectors of
30  the system of skew-normal SDEs. The length of the vectors must be equal
31  to the number of components given by ncomp.
32  - References to timescale, sigmasq, and lambda, which denote the parameter
33  vectors to be initialized based on timescale_, sigmasq_, and lambda_.
34 
35  - Must define the static function _type()_, returning the enum value of the
36  policy option. Example:
37  \code{.cpp}
38  static ctr::CoeffPolicyType type() noexcept {
39  return ctr::CoeffPolicyType::CONSTANT;
40  }
41  \endcode
42  which returns the enum value of the option from the underlying option
43  class, collecting all possible options for coefficients policies.
