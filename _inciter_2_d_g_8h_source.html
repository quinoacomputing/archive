2 /*!
3  \file src/Inciter/DG.h
4  \copyright 2016-2018, Los Alamos National Security, LLC.
5  \brief DG advances a system of PDEs with the discontinuous Galerkin scheme
6  \details DG advances a system of partial differential equations (PDEs) using
7  discontinuous Galerkin (DG) finite element (FE) spatial discretization (on
8  tetrahedron elements) combined with Runge-Kutta (RK) time stepping.
9 
10  There are a potentially large number of DG Charm++ chares created by
11  Transporter. Each DG gets a chunk of the full load (part of the mesh)
12  and does the same: initializes and advances a number of PDE systems in time.
13 
14  The implementation uses the Charm++ runtime system and is fully
15  asynchronous, overlapping computation and communication. The algorithm
16  utilizes the structured dagger (SDAG) Charm++ functionality. The high-level
17  overview of the algorithm structure and how it interfaces with Charm++ is
18  discussed in the Charm++ interface file src/Inciter/dg.ci.
19 
20  #### Call graph ####
21  The following is a directed acyclic graph (DAG) that outlines the
22  asynchronous algorithm implemented in this class The detailed discussion of
23  the algorithm is given in the Charm++ interface file transporter.ci. On the
24  DAG orange fills denote global synchronization points that contain or
25  eventually lead to global reductions. Dashed lines are potential shortcuts
26  that allow jumping over some of the task-graph under some circumstances or
27  optional code paths (taken, e.g., only in DEBUG mode). See the detailed
28  discussion in dg.ci.
29  \dot
30  digraph "DG SDAG" {
31  rankdir="LR";
32  node [shape=record, fontname=Helvetica, fontsize=10];
33  OwnGhost [ label="OwnGhost"
34  tooltip="own ghost data computed"
35  URL="\ref inciter::DG::setupGhost"];
36  ReqGhost [ label="ReqGhost"
37  tooltip="all of ghost data have been requested from us"
38  URL="\ref inciter::DG::reqGhost"];
39  OwnGhost -> sendGhost [ style="solid" ];
40  ReqGhost -> sendGhost [ style="solid" ];
41  OwnSol [ label="OwnSol"
42  tooltip="own solution/unknown data computed"
43  URL="\ref inciter::DG::advance"];
44  ComSol [ label="ComSol"
45  tooltip="communicated (ghost) solution/unknown data received"
46  URL="\ref inciter::DG::comsol"];
47  OwnSol -> Solve [ style="solid" ];
48  ComSol -> Solve [ style="solid" ];
49  }
50  \enddot
51  \include Inciter/dg.ci
52 
53 */
