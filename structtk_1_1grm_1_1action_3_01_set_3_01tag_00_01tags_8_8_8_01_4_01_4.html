template<typename tag, typename... tags>
struct tk::grm::action< Set< tag, tags... > >

Put value in state at position given by tags without conversion.

This struct and its apply function are used as a functor-like wrapper for calling the set member function of the underlying grammar stack, tk::Control::set.
