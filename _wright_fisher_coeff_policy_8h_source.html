11  - Must define a _constructor_, which is used to initialize the SDE

13  \code{.cpp}
14  CoeffPolicyName(
15  tk::ctr::ncomp_type ncomp,
16  const std::vector< kw::sde_omega::info::expect::type >& omega_,
17  std::vector< kw::sde_omega::info::expect::type >& omega )
18  \endcode
19  where
20  - ncomp denotes the number of scalar components of the system of the
21  Wright-Fisher SDE.
22  - Constant references to omega_ which denote a vector of real values used
23  to initialize the parameter vector of the Wright-Fisher SDE. The length
24  of the vector must be equal to the number of components given by ncomp.
25  - Reference to omega which denote the parameter vector to be initialized
26  based on omega_.
27 
28  - Must define the static function _type()_, returning the enum value of the
29  policy option. Example:
30  \code{.cpp}
31  static ctr::CoeffPolicyType type() noexcept {
32  return ctr::CoeffPolicyType::CONSTANT;
33  }
34  \endcode
35  which returns the enum value of the option from the underlying option
36  class, collecting all possible options for coefficients policies.
