40 //! \details Associate RNGs to global-scope wrappers. Admittedly, this code is
41 //! ugly and looks stupid at first sight. However, this is a translation of
42 //! runtime information (user-selected RNGs) to compile-time information:
43 //! associating RNG ids from an enum class, tk::ctr::RNGType::value, to a
44 //! compile-time constant, underlying_type value, facilitating a different
45 //! pair of global-scope RNG wrappers (uniform and uniform_bits) with code
46 //! reuse. Note that uniform and uniform_bits wrappers must be global-scope as
47 //! they are used as external generators to TestU01. Templating them on the
48 //! id enables the compiler generate a different wrapper for a different RNG
49 //! facilitating simultaneous calls to any or all wrappers as they are unique
50 //! functions.
51 //! \author J. Bakosi
52 // *****************************************************************************
