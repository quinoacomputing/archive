2 /*!
3  \file src/Inciter/Refiner.h
4  \copyright 2016-2018, Los Alamos National Security, LLC.
5  \brief Mesh refiner for interfacing the mesh refinement library
6  \details Mesh refiner is a Charm++ chare array and is used to interface the
7  mesh refinement object which does not know about parallelization and thus the
8  distributed nature of the mesh it operates on, i.e., it operates on mesh
9  chunks. Thus it does not do parallel communication and also does not know
10  about global vs local IDs. Instead this Charm++ chare array is the one that
11  does all parallel computing aspects, i.e., communcation, and using the mesh
12  refiner object as a library.
13 */
