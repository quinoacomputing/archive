       3                 :            :   \file      src/DiffEq/MixMassFractionBeta.h
       4                 :            :   \author    J. Bakosi
       5                 :            :   \date      Wed 21 Dec 2016 01:20:02 PM MST
       6                 :            :   \copyright 2012-2015, Jozsef Bakosi, 2016, Los Alamos National Security, LLC.
       7                 :            :   \brief     System of mix mass-fraction beta SDEs
       8                 :            :   \details   This file implements the time integration of a system of stochastic
       9                 :            :     differential equations (SDEs) with linear drift and quadratic diagonal
      10                 :            :     diffusion, whose invariant is the joint [beta
      11                 :            :     distribution](http://en.wikipedia.org/wiki/Beta_distribution). There are two
      12                 :            :     differences compared to the plain beta SDE (see DiffEq/Beta.h):
      13                 :            : 
      14                 :            :     - First, the parameters, b, and kappa are specified via functions that
      15                 :            :     constrain the beta SDE to be consistent with the turbulent mixing process.
      16                 :            :     In particular, the SDE is made consistent with the no-mix and fully mixed
      17                 :            :     limits. See, e.g., MixMassFractionBetaCoeffConst::update().
      18                 :            : 
      19                 :            :     - Second, there two additional random variables computed, the same as also
      20                 :            :     computed by the mass-fraction beta equation, see also
      21                 :            :     DiffEq/MassFractionBeta.h.
      22                 :            : 
      23                 :            :     In a nutshell, the equation integrated governs a set of scalars,
      24                 :            :     \f$0\!\le\!Y_\alpha\f$, \f$\alpha\!=\!1,\dots,N\f$, as
      25                 :            :     \f[
      26                 :            :        \mathrm{d}Y_\alpha(t) = \frac{b_\alpha}{2}\left(S_\alpha - Y_\alpha\right)
      27                 :            :        \mathrm{d}t + \sqrt{\kappa_\alpha Y_\alpha(1-Y_\alpha)}
      28                 :            :        \mathrm{d}W_\alpha(t), \qquad \alpha=1,\dots,N
      29                 :            :     \f]
      30                 :            :     with parameter vectors \f$b_\alpha = \Theta b'_\alpha > 0\f$, \f$
      31                 :            :     \newcommand{\irv}[1]{\langle{#1^2}\rangle} \kappa_\alpha = \kappa' \irv{x} >
      32                 :            :     0\f$, and \f$0 < S_\alpha < 1\f$. This is similar to DiffEq/Beta.h, but the
      33                 :            :     parameters, \f$b\f$ and \f$\kappa\f$ constrained. Here \f$
      34                 :            :     \newcommand{\irv}[1]{\langle{#1^2}\rangle}
      35                 :            :     \newcommand{\irmean}[1]{{\langle{#1}\rangle}} \Theta = 1 - \irv{x} /
      36                 :            :     [ \irmean{Y} (1-\irmean{Y}) ]\f$. The fluctuation about the mean, \f$
      37                 :            :     \newcommand{\irmean}[1]{{\langle{#1}\rangle}} \irmean{Y} \f$, is defined as
      38                 :            :     usual: \f$ \newcommand{\irmean}[1]{{\langle{#1}\rangle}} x = Y - \irmean{Y}
      39                 :            :     \f$, and \f$b'\f$ and \f$ \kappa'\f$ are user-specified constants. Also,
      40                 :            :     \f$\mathrm{d}W_\alpha(t)\f$ is an isotropic vector-valued
      41                 :            :     [Wiener process](http://en.wikipedia.org/wiki/Wiener_process) with
      42                 :            :     independent increments. The invariant distribution is the joint beta
      43                 :            :     distribution. This system of SDEs consists of N independent equations. For
      44                 :            :     more on the beta SDE, see http://dx.doi.org/10.1080/14685248.2010.510843.
      45                 :            : 
      46                 :            :     In addition to integrating the above SDE, there are two additional functions
      47                 :            :     of \f$ Y_\alpha \f$ are computed as
      48                 :            :     \f[ \begin{align}
      49                 :            :       \rho(Y_\alpha) & = \frac{ \rho_{2\alpha} }{ 1 + r_\alpha Y_\alpha } \\
      50                 :            :       V(Y_\alpha) & = \frac{1}{ \rho(Y_\alpha) }
      51                 :            :     \end{align} \f]
      52                 :            :     These equations compute the instantaneous mixture density, \f$ \rho \f$, and
      53                 :            :     instantaneous specific volume, \f$ V_\alpha \f$, for equation \f$ \alpha \f$
      54                 :            :     in the system. These quantities are used in binary mixing of
      55                 :            :     variable-density turbulence between two fluids with constant densities, \f$
      56                 :            :     \rho_1, \f$ and \f$ \rho_2 \f$. The additional parameters, \f$ \rho_2 \f$
      57                 :            :     and \f$ r \f$ are user input parameters and kept constant during
      58                 :            :     integration. Since we compute the above variables, \f$\rho,\f$ and \f$V\f$,
      59                 :            :     and call them mixture density and specific volume, respectively, \f$Y\f$,
      60                 :            :     governed by the beta SDE is a mass fraction.
      61                 :            : 
      62                 :            :     _All of this is unpublished, but will be linked in here once published_.
      63                 :            : */
