Detailed Description

template<class ProxyElem>
struct tk::Idx< ProxyElem >

Functor to dereference operator[] of chare proxy inside a variant

Since the chare array proxy is behind a variant, the returning element proxy from operator() is also a variant, defined by ProxyElem with a type depending on the input proxy, given by P, i.e., overloaded for all proxy types the variant supports.
