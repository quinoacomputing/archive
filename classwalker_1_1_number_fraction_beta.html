Detailed Description

template<class Init, class Coefficients>
class walker::NumberFractionBeta< Init, Coefficients >

NumberFractionBeta SDE used polymorphically with DiffEq.

The template arguments specify policies and are used to configure the behavior of the class. The policies are:

  • Init - initialization policy, see DiffEq/InitPolicy.h
  • Coefficients - coefficients policy, see DiffEq/NumberFractionBetaCoeffPolicy.h
