Detailed Description

Physics configurations for a system of transport equations.

Author
J. Bakosi
Copyright
2012-2015, Jozsef Bakosi, 2016, Los Alamos National Security, LLC.

This file defines policy classes for transport equations, defined in PDE/Transport.h.

General requirements on transport equation physics policy classes:

  • Must define the static function type(), returning the enum value of the policy option. Example:
    static ctr::PhysicsType type() noexcept {
    return ctr::PhysicsType::Advection;
    }
    which returns the enum value of the option from the underlying option class, collecting all possible options for coefficients policies.
