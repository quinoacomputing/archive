Detailed Description

CompFlow system of PDEs problem: Taylor-Green

See also
G.I. Taylor, A.E. Green, "Mechanism of the Production of Small Eddies from Large Ones", Proc. R. Soc. Lond. A 1937 158 499-521; DOI: 10.1098/rspa.1937.0036. Published 3 February 1937
Waltz, et. al, "Verification of a three-dimensional unstructured finite element method using analytic and manufactured solutions", Computers and Fluids, 2013, Vol.81, pp.57-67.

Query Dirichlet boundary condition value on a given side set for all components in this PDE system.

Returns
Vector of pairs of bool and boundary condition value associated to mesh node IDs at which Dirichlet boundary conditions are set. Note that instead of the actual boundary condition value, we return the increment between t+dt and t, since that is what the solution requires as we solve for the soution increments and not the solution itself.
