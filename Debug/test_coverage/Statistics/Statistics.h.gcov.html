      17                 :            :       x = X - \<X\>

      19                 :            :     - Letters can be augmented by a field ID, i.e., X2 is the full variable of
      20                 :            :       the second component of the vector X, while x1 = X1 - \<X1\> is the
      21                 :            :       fluctuation about the mean of the first component of vector X.
      22                 :            : 
      23                 :            :     - If the field ID is unspecified, it defaults to the first field, i.e.,
      24                 :            :       X = X1, x = x1, etc.
      25                 :            : 
      26                 :            :     - Statistical moments of arbitrary-length products can be computed.
      27                 :            : 
      28                 :            :       Examples:
      29                 :            :       - \<X\> - mean,
      30                 :            :       - \<xx\> - variance,
      31                 :            :       - \<xxx\> - third central moment,
      32                 :            :       - \<xy\> - covariance of X and Y,
      33                 :            :       - \<x1y2\> - covariance of the first component of vector X and the second
      34                 :            :         component of vector Y
      35                 :            : 
      36                 :            :     - In general, arbitrary-length products can be estimated that make up a
      37                 :            :       statistical moment, using any number and combinations of upper and
      38                 :            :       lower-case letters and their field IDs \< [A-Za-z][1-9] ... \>.
      39                 :            : 
      40                 :            :     - A statistical moment is ordinary if and only if all of its terms are
      41                 :            :       ordinary. A central moment has at least one term that is central, i.e., a
      42                 :            :       fluctuation about its mean.
      43                 :            : 
      44                 :            :       - Examples of ordinary moments: \<X\>, \<XX\>, \<XYZ\>, etc.
      45                 :            :       - Examples of central moments: \<x1x2\>, \<Xy\>, \<XYz\>, etc.
      46                 :            :

      48                 :            :       space variables.
