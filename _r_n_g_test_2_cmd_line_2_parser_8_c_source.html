62  // Reset parser's output stream to that of print's. This is so that mild
63  // warnings emitted during parsing can be output using the pretty printer.
64  // Usually, errors and warnings are simply accumulated during parsing and
65  // printed during diagnostics after the parser has finished. However, in some
66  // special cases we can provide a more user-friendly message right during
67  // parsing since there is more information available to construct a more
68  // sensible message. This is done in e.g., tk::grm::store_option. Resetting
69  // the global g_print, to that of passed in as the constructor argument allows
70  // not to have to create a new pretty printer, but use the existing one.
71  tk::grm::g_print.reset( print.save() );

75  tao::pegtl::parse< cmd::read_string, tk::grm::action >( in, cmd );
76 
77  // Echo errors and warnings accumulated during parsing
