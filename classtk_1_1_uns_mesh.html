std::size_t tk::UnsMesh::m_graphsize
private

Number of nodes

Stores the size (number of nodes) of the mesh graph. Used if only the graph is needed but not the node coordinates, e.g., for graph partitioning, in which case only the connectivity is required. If the coordinates are also loaded, the member functions nnode() and size() return the same.
