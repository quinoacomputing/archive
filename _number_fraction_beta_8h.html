This file implements the time integration of a system of stochastic differential equations (SDEs) with linear drift and quadratic diagonal diffusion, whose invariant is the joint beta distribution. The main difference compared to the plain beta SDE (see DiffEq/Beta.h), is that in the number-fraction beta SDE the dependent variable, there are two additional stochastic variables computed from the beta variables.
