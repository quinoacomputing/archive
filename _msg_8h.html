Detailed Description

Custom Charm++ message types definitions.

2012-2015, J. Bakosi, 2016-2018, Los Alamos National Security, LLC.

This file defines some custom Charm++ message types. This is somewhat experimental, not well tested, and is not currently in use. The best is to avoid this kind of low-level messages as this can be error-prone.
