Grammar definition for parsing the command line. We use the Parsing Expression Grammar Template Library (PEGTL) to create the grammar and the associated parser. Credit goes to Colin Hirsch (pegtl.nosp@m.@coh.nosp@m.i.at) for PEGTL. Word of advice: read from the bottom up.
