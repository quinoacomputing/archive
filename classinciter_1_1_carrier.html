void Carrier::bc ( )
private

Extract node IDs from side set node lists and match to.

Boundary conditions (BC), mathematically speaking, are applied on finite surfaces. These finite surfaces are given by element sets. This function queries the node lists associated to side set IDs as read in from file (old mesh node IDs as in file associated to all side sets in file). Then the user-specified boundary conditions, their values and which side set they are assigned to, are interrogated and only those nodes and their BC values are extracted that we operate on. This BC data structure is then sent to the linear system merger which needs to know about this to apply BCs before a linear solve. Note that the BC mesh nodes that this function results in, stored in dirbc and sent to the linear system merger, only contains those nodes that this chare contributes to, i.e., it does not contain those BC nodes at which other chares enforce Dirichlet BCs. The linear system merger then collects these and communicates to other PEs so that BC data held in LinSysMerger::m_bc are the same on all PEs. That is the authoritative BC data, which can be queried by LinSysMerger::bc().
