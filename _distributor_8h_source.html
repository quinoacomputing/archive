12  high-level overview of the algorithm structure and how it interfaces with
13  Charm++ is discussed in the Charm++ interface file
14  src/Walker/distributor.ci.

19  the algorithm is given in the Charm++ interface file distributor.ci,

21  fills denote global synchronization points, orange frames with white fill
22  are partial synchronization points that overlap with other tasks, and dashed
23  lines are potential shortcuts that allow jumping over some of the task-graph
