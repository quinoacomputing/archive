Detailed Description

Random number generator test harness.

This file defines a generic random number generator test harness class. The class uses runtime polymorphism without client-side inheritance: inheritance is confined to the internals of the class, inivisble to client-code. The class exclusively deals with ownership enabling client-side value semantics. Credit goes to Sean Parent at Adobe: https://github.com/sean-parent/ sean-parent.github.com/wiki/Papers-and-Presentations.
